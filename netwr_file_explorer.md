# === netwr file explorer ===

## file operation

- %     |  create a file
- R     |  rename a file
- mf    |  mark a file
- D     |  delete target
- d     |  create dir

## navigation

- gh    |  toggle hidden file
- p     |  preview the file
- C-w z |  close preview

## mark file operation

- mt    |  target dir
- mf    |  mark a file
- mc    |  copy mf
- mm    |  move mf
- mx    |  external cmd
