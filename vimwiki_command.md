# === Vim wiki command ===

| command                | action                               |
|------------------------|--------------------------------------|
| :Vimwiki2HTML          | Convert current wiki link to HTML.   |
| :VimwikiAll2HTML       | Convert all your wiki links to HTML. |
| :help vimwiki-commands | List all commands.                   |
| :help vimwiki          | General vimwiki help docs.           |
