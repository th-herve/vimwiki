# Qtile Keybind

| key             | action                   |
|-----------------|--------------------------|
| mod+j/k         | previous/next pane       |
| mod+space       | switch pane              |
| mod+tab         | switch layout            |
| mod+w           | close window             |
| mod+ctrl+r      | reload config            |
| mod+enter       | open terminal            |
| mod+r           | start app                |
| mod+(group num) | swith to group           |
| mod+shift+group | send window to the group |
