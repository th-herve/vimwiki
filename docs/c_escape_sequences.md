# C escape sequences


| sequences | char                   |
|-----------|------------------------|
| \a        | alert bell             |
| \\        | \                      |
| \b        | backspace              |
| \?        | ?                      |
| \f        | formfeed (page break)  |
| \n        | newline                |
| \'        | '                      |
| \"        | "                      |
| \"        | "                      |
| \r        | return                 |
| \ 000     | octal number           |
| \x hh     | hexa number            |
| \t        | horizontal tab         |
| \v        | vertical tab           |
| \0        | null character         |
