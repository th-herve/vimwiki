## Access modifiers

Four types:

| modifier    | class | package | child class | global |
|-------------|-------|---------|-------------|--------|
| public      | ✓     | ✓       | ✓           | ✓      |
| protected   | ✓     | ✓       | ✓           |        |
| no modifier | ✓     | ✓       |             |        |
| private     | ✓     |         |             |        |
