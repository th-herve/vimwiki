# Media query

```css
body {
  margin: 24px;
}

@media (max-width: 600px) {
  body {
    margin: 8px;
  }
}
```
## Common breakpoint

- Mobile: <500px
- Tablets: 500px - 1000px
- Browser: >1000px
- Wide screen: >2000px

[More on it](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_media_queries/Using_media_queries)
