# Javascript arrays

## Create an array

const [array_name] = [item1, item2...];

```javascript
const cars = ["Saab", "Volvo", "Bmw"];
```
>It is common practice to declare arrays with the `const` keyword.

## Accessing array elements

Use `[]`

```javascript
let car = cars[0];

cars[1] = "Opel";
```
