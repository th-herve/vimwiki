# Pacman

| cmd        | description             |
|------------|-------------------------|
| -S (pkg)   | install                 |
| -R (pkg)   | remove                  |
| -Rs (pkg)  | remove + dependency     |
| -Syu (pkg) | Update all pkg          |
| -Ss (pkg)  | search                  |
| -Qs (pkg)  | search in installed pkg |

> Single pkg can be updated with -Sy, but it should be avoided.
