# Java loops

## For

```java
for (int i = 0; i < 10; i++) {
    // code
}
```

## While

```java
int num = 0;
while (num < 20) {
    num++;
}
```

## Do-while

```java
int num = 0;
do {
    num++
} while (num < 4);
```

## For each

```java
for (String s : myArray) {
    System.out.println(s);
}
```
