# Java getter and setter

## Accessor methods

```java
public class Cat {
    private String name = "Jitsi";
    
    // getter
    public String getName() {
        return name;
    }
}

myCat.getName();
```

## Mutator methods

```java
public class cat {
    private string name = "Jitsi";
    
    public void setName(String newName) {
        this.name = newName;
    }
}
```
