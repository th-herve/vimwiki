# C arrays


```c
int num[] = {1, 2, 3, 4}; // the size is automatically set to the number of value initialized

int nbr[10] = {1, 2, 3, 4} // if a size is specified, and there is less value initialized, other elements are set to 0


char str = "boo";
// is a shorthand for:
char str[] = {'b', 'o', 'o', '\0'};
```
