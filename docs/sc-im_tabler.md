# sc-im

Tabler in cli for linux.

## Basics

| key    | action            |
|--------|-------------------|
| \      | insert string     |
| E      | edit string       |
| =      | insert int        |
| e      | edit int          |
| x      | delete cell       |
| C-j    | auto re size cell |
| goab12 | go to cell ab12   |
| yy     | yank cell         |
| u/C-r  | undo/redo         |
| ir/ic  | insert row/col    |
| dr/dc  | delete row/col    |

## Command

| command             | action      | example |
|---------------------|-------------|---------|
| :cellcolor "bold=1" | set in bold |         |
